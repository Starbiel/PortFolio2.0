Tecnologias que trabalho

Tecnologia: HTML

O HTML é a linguagem padrão para criação e estruturação de documentos na web. Ele consiste em uma série de elementos ou tags que são usados para definir a estrutura e o conteúdo de uma página da web.

Na hora de estruturar a pagina, eu utilizo do maximo de tags possiveis, para que seja mantido uma semantica e fácil vizualização

Conhecimento 10/10

Tecnologia:CSS

O css é usada para controlar a aparência e o layout de elementos em documentos HTML. Ele trabalha em conjunto com o HTML para definir o design, cores, fontes, espaçamentos e outras propriedades visuais de uma página da web.

Na hora de estruturar a pagina, eu utilizo do maximo de seus recursos, para gerar um sistema totalmente responsivo e dinâmico

Conhecimento 9/10

Tecnologia:JavaScript

JavaScript é uma habilidade valiosa para desenvolvedores web e é uma das linguagens mais utilizadas na indústria de tecnologia atualmente.

Atualmente, eu utilizo para criar lógica e interatividade nas paginas, também utilizo alguns frameworks como Node

Conhecimento 8/10

Tecnologia:PHP

Ela permite a criação de páginas dinâmicas e interativas, processamento de formulários, acesso a bancos de dados e muito mais.

Atualmente, eu utilizo alguns frameworks como Laravel.
